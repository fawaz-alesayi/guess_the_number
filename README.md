### Guess The Number
A simple Rust CLI program that generates a random number and lets you guess it.

### Usage
```bash
cargo run
```

```
A number has been between 0 and 100 has been generated. Guess the number!
15 <-- You enter this into the CLI
Too low!
19
Too low!
95
Too high!
50
Too low!
70
Bingo!
```
